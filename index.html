Quick Setup Guide

Add RoxTether to Your Server

Invite the bot with permissions for Manage Roles, Send Messages, and Slash Commands.

Configure Group Bindings

Use /bind [group_id] [min_rank] [discord_role] to link Roblox groups to Discord roles.

Members Verify Accounts

Users run /verify to securely link accounts via Roblox OAuth.

Roles are assigned instantly and updated automatically as statuses change.
